# Punchlister

A construction site management web app built for project managers and site supervisors. It centralises the day-to-day admin of a construction project into one tool.

## What it does

**Field Logs** — capture voice notes, photos, or typed observations on site. AI automatically summarises entries, flags issues, and suggests follow-up actions.

**Punch List** — track outstanding tasks per project. Items can be extracted automatically from field logs or added manually, with assignee, priority, and category.

**RFI Manager** — raise and track Requests for Information. RFIs can be generated directly from flagged field log entries.

**Variations** — log and manage contract variations with status tracking and cost estimates.

**Daily Reports** — generate structured daily site reports from field log activity.

**Subcontractor Tracker** — keep a live overview of which trades are on site, crew sizes, and work areas.

## Tech stack

- React + Vite
- Supabase (auth + database)
- Tailwind CSS + Framer Motion
- Claude AI (field log analysis and action extraction)
